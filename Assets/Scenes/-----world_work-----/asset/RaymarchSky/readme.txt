■概要
オブジェクトの中に入ったときにSkyboxを切り替えるシェーダーです。
本シェーダーはレイマーチングにより巨大な壁を生成し、Skyboxのテクスチャを貼り付けることで実装されています。

以前公開した「中に入るとSkyboxが変わるシェーダー(https://pya1234.booth.pm/items/2080309)（以下①と表記）」と以下の点で異なります。

(1)①は巨大なCubeオブジェクトを展開するため他オブジェクトに干渉する可能性がある。
　→本シェーダーは適用したオブジェクト内でSkyboxを描画するため他オブジェクトに干渉しません。

(2)①はSkyboxを切り替えるだけで他オブジェクトの表示はそのままとなる。
　→本シェーダーは適用したオブジェクトの外側にあるオブジェクトが見えなくなります。
　　つまり、オブジェクト内部にカメラが入ったときSkybox以外見えなくなります。

(3)①はSkyboxをCubemapで指定する。
　→6 Sided形式で指定するように変更しました。

(4)①は2種類のシェーダーを組み合わせる必要がある
　→1つになりました。

■シェーダーの説明
　RaymarchSkb
　・Display Object
　　　オブジェクトの表示ON/OFF
　　　OFFにした場合、オブジェクト内部にカメラが入った時のみSkyboxを描画します
　・6 Sided
　　　Skyboxのテクスチャを指定
　・SkyBox Scale
　　　Skyboxのサイズ
　　　値を大きくすると負荷が上がります
　　　
　本シェーダーを適用したオブジェクトのスケールを変更する場合、XYZを同じ値にしないとSkyboxの一部が欠けて表示されるケースがあります。
　その場合は以下の方法を試してください。
　(1)Skybox Scaleのパラメータ値を下げる
　(2)RaymarchSkb.shaderに定義されているRAY_LOOPの回数を上げる。ただし負荷が上昇しFPSが低下する可能性があります。
　(3)XYZの値を統一する（もしくは近い値にする）
　

■動作確認環境
Unity 2018.4.20f1
